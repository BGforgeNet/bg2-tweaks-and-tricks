Zbiór usprawnień, które poprawiają irytujące i/lub nierealistyczne elementy gry, dodają trochę klimatu, ułatwiają pewne rzeczy, a inne czynią trudniejszymi.

_Prawdopodobnie nie wszystkie komponenty są dla każdego, ale niemal każdy znajdzie tu coś dla siebie._

Mod działa z każdą grą na silniku Infinity Engine, ale nie wszystkie komponenty mają zastosowanie we wszystkich grach.

## Instalacja

- Pobierz plik ZIP ze [strony najnowszego wydania](https://github.com/BGforgeNet/bg2-tweaks-and-tricks/releases/latest).
- Wypakuj go do głównego katalogu gry.
- Uruchom `setup-tnt.exe`.
- Zainstaluj wybrane komponenty.
